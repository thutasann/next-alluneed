# NEXTJS-ECOMMERCE

This is the Ecommerce Web app including Admin Dashboard

## Tech Stacks

- nextjs (app dir)
- prisma
- mongodb
- clerk
- shadcn-ui
- tailwind

## Preview URL:

- [Admin](https://alluneed-admin.vercel.app/)
- [Store](https://alluneed.vercel.app/)

## Run Locally

### Install the Dependencies

```bash
cd admin

yarn
```

```bash
cd store

yarn
```

### Start the Servers

```bash
cd admin

yarn dev
```

```bash
cd store

yarn dev
```
